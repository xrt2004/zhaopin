提示：

过多的项目设置邮件通知会消耗系统资源。

邮件发送规则
注册会员：      (注册会员成功后发送邮件通知)
申请职位：      (个人申请了职位是否通知企业)
邀请面试：      (企业邀请个人面试是否通知个人)
新增订单：      (会员下充值订单是否邮件通知)
付款成功：      (会员订单付款完成是否邮件通知)
修改密码：      (会员修改密码后是否邮件通知)
职位审核通过：      (职位审核通过是否邮件通知)
职位审核未通过：      (职位审核未通过是否邮件通知)
营业执照审核通过：      (营业执照审核通过是否邮件通知)
营业执照审核未通过：      (营业执照审核未通过是否邮件通知)
简历审核通过：      (个人简历审核通过是否邮件通知)
简历审核未通过：      (个人简历审核未通过是否邮件通知)
简历头像审核通过：      (个人简历头像审核通过是否邮件通知)
简历头像审核未通过：      (个人简历头像审核未通过是否邮件通知)
